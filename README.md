# Yad2-Client

### Description
A replica of Yad2 website, includes:
- REST API (Node.JS and Express) 
- DB in MongoDB
- (Also transfered to a DB in MSSQL) 
- User management and editing
- Adding a new post
- Viewing and interacting with post
- Advanced search 
- Media in S3 bucket

### Technologies
- React
- Node JS
- MongoDB
- SQL
- AWS S3
- Sass
- HTML
- JS
